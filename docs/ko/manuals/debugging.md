# 디버깅

Defold에는 검사 기능을 갖춘 통합 Lua 디버거가 있습니다. 내장된 [프로파일링 도구](/manuals/profiling)와 함께 사용하면 게임 로직의 버그 원인을 찾거나 성능 문제를 분석하는 데 도움이 되는 강력한 도구입니다. 또한 엔진 자체가 드물게 크래시되는 경우를 위해 크래시 로그도 제공합니다.

* [게임 로직 디버깅](/manuals/debugging-game-logic)에 대해 더 알아보세요.
* [네이티브 코드 디버깅](/manuals/debugging-native-code)에 대해 더 알아보세요.

# 애니메이션

Defold에는 컴포넌트의 그래픽 소스로 사용할 수 있는 여러 종류의 애니메이션에 대한 내장 지원이 있습니다:

* [플립북 애니메이션](/manuals/flipbook-animation) - 정지 이미지 시퀀스를 연속으로 재생합니다
* [모델 애니메이션](/manuals/model-animation) - 3D 스킨드 애니메이션을 재생합니다
* [프로퍼티 애니메이션](/manuals/property-animation) - 위치, 스케일, 회전 등 여러 프로퍼티에 애니메이션을 적용합니다

추가 애니메이션 형식은 익스텐션을 통해 추가할 수 있습니다:

* [Rive 애니메이션](/extension-rive) - 벡터 기반 2D 스켈레탈 애니메이션을 재생합니다
* [Spine 애니메이션](/extension-spine) - 텍스쳐 기반 2D 스켈레탈 애니메이션을 재생합니다

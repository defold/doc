# Group과 Mask

물리 엔진에서는 충돌 오브젝트를 그룹화하고, 서로 어떻게 충돌할지 필터링할 수 있습니다. 이는 이름이 있는 _충돌 그룹_으로 처리됩니다. 생성하는 각 충돌 오브젝트마다 두 프로퍼티인 *Group*과 *Mask*가 해당 오브젝트가 다른 오브젝트와 충돌하는 방식을 제어합니다.

두 오브젝트 사이의 충돌이 등록되려면 두 오브젝트 모두 각자의 *Mask* 필드에 서로의 그룹을 지정해야 합니다.

![물리 충돌 그룹](images/physics/collision_group.png)

*Mask* 필드에는 여러 그룹 이름을 포함할 수 있으므로 복잡한 상호작용 시나리오를 만들 수 있습니다.

## 충돌 감지
그룹과 마스크가 서로 일치하는 두 충돌 오브젝트가 충돌하면 물리 엔진은 게임에서 충돌에 반응하는 데 사용할 수 있는 [충돌 메세지](/manuals/physics-messages)를 생성합니다.

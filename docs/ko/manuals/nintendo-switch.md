# Nintendo Switch 개발

Nintendo 라이선스 제한으로 인해 Nintendo Switch 플랫폼을 지원하는 Defold 버전에 대한 액세스는 Defold의 표준 버전에 포함되어 있지 않습니다. Nintendo Switch 지원이 포함된 Defold 버전에 액세스하려면 Nintendo Switch의 승인된 게임 개발자가 되어야 합니다.


## Nintendo Switch 개발자로 등록하기

[Nintendo Developer Portal](https://developer.nintendo.com/register)에서 Nintendo Switch 게임 개발자로 등록할 수 있습니다:

![](images/nintendo-switch/register-nintendo.png)

Nintendo의 승인을 받으면 Nintendo Developer Portal의 Tools and Middleware 페이지에 액세스할 수 있으며, 여기에서 Defold 액세스를 신청할 수 있습니다. Defold 액세스를 신청하면 Nintendo는 해당 신청자가 등록된 Nintendo 개발자임을 확인하는 이메일을 Defold 팀에 보냅니다.


## Defold의 Nintendo Switch 액세스

승인된 Nintendo Switch 개발자 상태가 확인되면 다음에 대한 액세스를 제공합니다:

* 콘솔별 API 통합이 포함된 Nintendo Switch 익스텐션의 소스 코드 액세스.
* Nintendo Switch를 지원하는 Defold 게임엔진 버전의 소스 코드 액세스. Nintendo Switch용 게임을 빌드하는 데 소스 코드 액세스가 필수는 아니지만, 엔진 코어에 소스 코드 기여를 하고 싶은 경우를 위해 액세스를 제공합니다.
* Nintendo Switch 플랫폼으로 번들링할 수 있는 [커맨드 라인 도구](/manuals/bob). Defold 에디터에서의 번들링은 지원되지 않습니다.
* Nintendo Switch 관련 지원을 받을 수 있는 포럼.

# 광고

광고는 웹과 모바일 게임에서 수익을 창출하는 매우 일반적인 방법이 되었으며, 수십억 달러 규모의 산업으로 성장했습니다. 개발자는 게임에 표시한 광고를 시청한 사람 수에 따라 수익을 받습니다. 보통 시청자가 많을수록 수익도 많아지는 단순한 구조이지만, 실제 지급액에는 다른 요인도 영향을 줍니다.

* 광고 품질 - 관련성 높은 광고는 플레이어의 상호작용과 관심을 끌 가능성이 더 높습니다.
* 광고 형식 - 배너 광고는 보통 지급액이 낮고, 처음부터 끝까지 시청한 전체 화면 광고는 지급액이 더 높습니다.
* 광고 네트워크 - 지급액은 광고 네트워크마다 다릅니다.

::: sidenote
CPM = Cost per mille. 광고주가 1,000회 조회당 지불하는 금액입니다. CPM은 광고 네트워크와 광고 형식에 따라 달라집니다.
:::

## 형식

게임에서 사용할 수 있는 광고 형식은 다양합니다. 가장 흔한 형식으로는 배너, 인터스티셜, 보상형 광고가 있습니다.

### 배너 광고

배너 광고는 텍스트, 이미지 또는 비디오 기반이며 화면의 비교적 작은 영역을 차지합니다. 보통 화면 상단이나 하단에 표시됩니다. 배너 광고는 구현하기 매우 쉽고, 광고 영역을 쉽게 예약할 수 있는 캐주얼 단일 화면 게임에 잘 맞습니다. 배너 광고는 사용자가 게임을 중단 없이 플레이하는 동안 노출을 극대화합니다.

### 인터스티셜 광고

인터스티셜 광고는 애니메이션과 때로는 상호작용 가능한 *리치 미디어* 컨텐츠를 포함하는 큰 전체 화면 경험입니다. 인터스티셜 광고는 게임 경험에서 자연스럽게 쉬어가는 지점인 레벨 사이 또는 게임 세션 사이에 주로 표시됩니다. 인터스티셜 광고는 일반적으로 배너 광고보다 조회 수가 적지만 비용(CPM)이 배너 광고보다 훨씬 높아서 전체 광고 수익에 큰 영향을 줄 수 있습니다.

### 보상형 광고

보상형 광고(Incentivized ads라고도 함)는 선택 사항이므로 다른 여러 광고 형식보다 방해가 덜 됩니다. 보상형 광고는 보통 인터스티셜 광고처럼 전체 화면 경험입니다. 사용자는 광고를 시청하는 대가로 *전리품*, 코인, 생명, 시간 또는 기타 게임 내 화폐나 혜택 같은 보상을 선택할 수 있습니다. 보상형 광고는 일반적으로 비용(CPM)이 가장 높지만, 조회 수는 사용자의 참여 선택 비율에 직접적으로 좌우됩니다. 보상이 충분히 가치 있고 적절한 시점에 제공될 때에만 보상형 광고는 뛰어난 성과를 낼 수 있습니다.


## 광고 네트워크

[Defold Asset Portal](/tags/stars/ads/)에는 광고 제공업체와 연동되는 여러 에셋이 있습니다.

* [AdMob](https://defold.com/assets/admob-defold/) - Google AdMob 네트워크를 사용해 광고를 표시합니다.
* [Enhance](https://defold.com/assets/enhance/) - 여러 광고 네트워크를 지원합니다. 추가 post-build 단계가 필요합니다.
* [Facebook Instant Games](https://defold.com/assets/facebookinstantgames/) - Facebook Instant Game에서 광고를 표시합니다.
* [IronSource](https://defold.com/assets/ironsource/) - IronSource Ad network를 사용해 광고를 표시합니다.
* [Unity Ads](https://defold.com/assets/defvideoads/) - Unity Ads 네트워크를 사용해 광고를 표시합니다.


# 게임에 광고 통합하기

게임에 통합할 광고 네트워크를 정했다면 해당 *에셋*의 설치 및 사용 지침을 따라야 합니다. 일반적으로 먼저 익스텐션을 [프로젝트 종속성](/manuals/libraries/#setting-up-library-dependencies)으로 추가합니다. 에셋을 프로젝트에 추가한 뒤에는 통합을 진행하고 해당 에셋 전용 함수를 호출해 광고를 로드하고 표시할 수 있습니다.


# 광고와 인앱 구매 결합하기

모바일 게임에서는 광고를 영구적으로 제거하기 위해 [인앱 구매](/manuals/iap)를 제공하는 경우가 꽤 흔합니다.

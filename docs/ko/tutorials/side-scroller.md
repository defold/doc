# 사이드 스크롤러 튜토리얼


이 튜토리얼은 Defold 에디터에 통합되어 있어 쉽게 접근할 수 있습니다.

Defold를 시작하고:
1. 왼쪽에서 `Create From` ▸ `TUTORIALS`를 선택합니다.
2. "Side Scroller Tutorial"을 선택합니다.
3. 원한다면 이름을 바꿉니다.
4. 로컬 드라이브에서 프로젝트 위치를 선택합니다.
5. `Create New Project`를 클릭합니다.

![new project](images/new-side-scroller.webp)

에디터는 전체 튜토리얼 텍스트가 들어 있는 프로젝트 루트의 "README" 파일을 자동으로 엽니다.

![icon](images/icon-tutorial.svg){.icon} [GitHub에서도 전체 튜토리얼 텍스트를 읽을 수 있습니다](https://github.com/defold/tutorial-side-scroller)

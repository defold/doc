# Colorslide 튜토리얼

이 중간 난이도 튜토리얼에서는 간단한 다중 레벨 모바일 게임을 위한 게임 내 GUI, 레벨 선택 GUI 화면, 시작 화면을 만듭니다.

이 튜토리얼은 Defold 에디터와 통합되어 있으며 쉽게 접근할 수 있습니다.

1. Defold를 시작합니다.
2. 왼쪽에서 *New Project*를 선택합니다.
3. *From Tutorial* 탭을 선택합니다.
4. "Colorslide tutorial"을 선택합니다.
5. 로컬 드라이브에서 프로젝트 위치를 선택하고 *Create New Project*를 클릭합니다.

![new project](images/new-colorslide.png)

에디터가 프로젝트 루트에 있는 "README" 파일을 자동으로 엽니다. 이 파일에는 전체 튜토리얼 텍스트가 들어 있습니다.

![icon](images/icon-tutorial.svg){.icon} [전체 튜토리얼 텍스트는 GitHub에서도 읽을 수 있습니다](https://github.com/defold/tutorial-colorslide)

막히는 부분이 있으면 [Defold Forum](//forum.defold.com)으로 오세요. Defold 팀과 친절한 많은 사용자에게 도움을 받을 수 있습니다.

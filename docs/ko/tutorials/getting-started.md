# 시작하기

Defold 엔진과 에디터는 배울 기능이 많은 강력한 도구입니다. 시작하는 데 도움이 되도록 다양한 튜토리얼을 준비했습니다. 많은 튜토리얼은 에디터 안에서 바로 사용할 수 있으므로 매우 쉽게 시작할 수 있습니다.

## 에디터에서 튜토리얼 시작하기

Defold 에디터를 실행하면 프로젝트 선택 및 생성 화면이 표시됩니다. 여기서 시도해 볼 튜토리얼을 쉽게 선택할 수 있습니다.

1. Defold를 시작합니다.
2. 왼쪽에서 *New Project*를 선택합니다.
3. *From Tutorial* 탭을 선택합니다.
4. 관심 있는 튜토리얼을 선택합니다.
5. 로컬 드라이브에서 프로젝트를 저장할 위치를 선택합니다.
6. *Create New Project*를 클릭합니다.

![프로젝트 생성](images/getting-started/new-project.png)

이제 에디터가 튜토리얼 프로젝트를 자동으로 다운로드하고, 에디터에서 연 다음 튜토리얼 텍스트(프로젝트 루트의 "README" 파일)를 엽니다.

![튜토리얼 텍스트](images/getting-started/tutorial-text.png)

이제 튜토리얼 텍스트를 따라 진행하세요! 텍스트로 돌아가려면 *Assets* 보기에서 "README" 파일을 <kbd>double click</kbd>하세요. 열린 파일 탭을 <kbd>right click</kbd>하고 <kbd>Move to Other Tab Pane</kbd>을 선택하면 작업 중인 파일과 튜토리얼 텍스트를 나란히 볼 수도 있습니다.

![나란히 보기](images/getting-started/side-by-side.png)

Defold를 처음 사용한다면 [에디터 소개](/manuals/editor)도 확인해 보세요.

막히는 부분이 있으면 [Defold Forum](//forum.defold.com)으로 이동하세요. Defold 개발자와 친절한 사용자들이 도움을 드립니다.

즐거운 Defolding 되세요!

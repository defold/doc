## 에디터 1과 2

현재 Defold 에디터 2로 전환하는 중이며, 에디터 2는 현재 베타 상태입니다. 새로 만드는 문서 대부분은 새 에디터를 기준으로 작성하며 결국 모든 문서를 업데이트할 예정이지만, 이 과정에는 시간이 걸립니다. 스크린샷의 색상 테마를 보고 에디터 버전을 구분할 수 있습니다.

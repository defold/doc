# 自顶向下移动教程

在这个初学者教程中，您将学习如何在 Defold 中创建一个简单的自顶向下角色控制器。您将从一个特别准备好的项目开始，因此无需担心资源和基础设置，而是专注于机制。

您将了解 Defold 项目的基本结构，包括集合、游戏对象、精灵、脚本、瓦片地图、图集和摄像机。您将为角色添加行走动画，处理玩家输入，使用 Lua 移动游戏对象，规范化对角线移动，并根据移动方向切换动画。

最后，您将得到一个可以在小型瓦片地图关卡中四处移动的动画角色，并准备好继续扩展 WASD 控制、摄像机跟随或更大的地图等功能。

## 直接从 Defold 运行

该教程已集成到 Defold 编辑器中，可从 Defold 欢迎界面轻松访问：

1. 在左侧选择 *Create From* -> <kbd>Tutorials</kbd>。
2. 选择 <kbd>Top-down Movement Tutorial</kbd>。
3. 输入项目的 *Title*。
4. 在本地驱动器上选择项目的 *Location*。
5. 单击 <kbd>Create New Project</kbd>。

![新建项目](images/top-down-start.webp)

编辑器会自动从项目根目录打开 "README" 文件，其中包含您可以跟随的完整教程文本。

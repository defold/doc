# Поддержка Xbox

Defold пока не поддерживает Microsoft Xbox. Фонд Defold активно работает над тем, чтобы стать поставщиком middleware для Xbox. Мы рекомендуем утвержденным разработчикам Xbox связаться со своим контактом в программе разработки Xbox и запросить поддержку Defold для Microsoft Xbox.

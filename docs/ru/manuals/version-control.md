# Контроль версий

Проекты Defold хорошо работают с [Git](https://git-scm.com), но синхронизация выполняется вне редактора. Используйте предпочитаемый Git-клиент или командную строку, чтобы клонировать репозиторий, получать и забирать изменения, создавать коммиты и ветки, отправлять изменения и разрешать конфликты.

## Измененные файлы 

Если каталог проекта является корнем рабочего дерева Git и содержит хотя бы один коммит, Defold отображает в панели редактора *Changed Files* неигнорируемые файлы, обнаруженные как добавленные, изменённые, удалённые или переименованные. Редактор определяет эти записи, сравнивая файлы на диске непосредственно с текущим коммитом (`HEAD`), поэтому добавление изменения в индекс не влияет на список. Разрешайте конфликты слияния во внешнем Git-клиенте.

![измененные файлы](images/workflow/changed_files.png)

Выберите ровно один изменённый или переименованный файл и нажмите <kbd>Diff</kbd>, чтобы просмотреть текстовые различия. Нажмите <kbd>Revert</kbd>, чтобы отменить выбранные изменения в рабочем дереве и индексе. Отслеживаемые файлы восстанавливаются до состояния `HEAD`; файлы, отсутствующие в `HEAD`, удаляются независимо от того, добавлены ли они в индекс; при переименовании новый путь удаляется, а старый восстанавливается. В редакторе отменить эту операцию невозможно, поэтому заранее создайте коммит или резервную копию работы, которая может понадобиться.

## Git

Git эффективно хранит текстовые файлы проектов Defold. Однако часто изменяющиеся крупные бинарные ресурсы, например PSD-файлы или проекты для обработки звука, могут быстро увеличивать историю репозитория. Для больших рабочих файлов стоит рассмотреть Git LFS либо отдельное хранилище и систему резервного копирования.

Панель *Changed Files* предоставляет только локальные операции просмотра состояния, различий и отката. Она не знает, были ли коммиты отправлены в удалённый репозиторий, и не выполняет fetch, pull, commit или push. Выполняйте эти операции во внешнем Git-клиенте или из командной строки. По умолчанию Defold перезагружает внешние изменения и обновляет панель при возвращении фокуса. Если параметр *Load External Changes on App Focus* отключён, выберите *File ▸ Load External Changes*.

# Defold SDK

Defold SDK содержит необходимую функциональность для объявления нативного расширения, а также взаимодействия с низкоуровневым нативным слоем платформы, на котором работает приложение, и высокоуровневым слоем Lua, в котором создается игровая логика.

## Использование

Расширения на C++ могут подключать общий заголовочный файл `dmsdk/sdk.h`:

```cpp
#include <dmsdk/sdk.h>
```

Общий заголовочный файл содержит объявления C++, поэтому его нельзя подключать из исходного файла C. Исходные файлы C должны подключать отдельные необходимые им заголовочные файлы `.h`, совместимые с C. Например:

```c
#include <dmsdk/extension/extension.h>
#include <dmsdk/dlib/configfile.h>
#include <dmsdk/resource/resource.h>
```

Сейчас только часть dmSDK имеет интерфейс на чистом C; не у каждой подсистемы C++ есть эквивалент на C. Доступные функции и типы описаны в [обзоре C API](/ref/overview_defoldc/) и [обзоре C++ API](/ref/overview_defoldcpp/). Заголовочные файлы Defold SDK включены в отдельный архив `defoldsdk_headers.zip` для каждого [релиза Defold на GitHub](https://github.com/defold/defold/releases). Эти заголовочные файлы можно использовать для автодополнения кода в выбранном редакторе.

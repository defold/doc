  `.`
  : Сокращение, вычисляющее текущий игровой объект.

  `#`
  : Сокращение, вычисляющее текущий компонент.

  Пример:

  ```lua
   -- Позволить игровому объекту получить фокус ввода
   msg.post(".", "acquire_input_focus")
  ```

  ```lua
   -- Передать "reset" текущему скрипту
   msg.post("#", "reset")
  ```

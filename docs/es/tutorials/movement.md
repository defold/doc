# Tutorial de movimiento

En este tutorial para principiantes aprenderás a crear movimiento realista usando vectores y algo de álgebra vectorial sencilla.

El tutorial está integrado con el editor Defold y es fácil acceder a él:

1. Inicia Defold.
2. Selecciona *New Project* a la izquierda.
3. Selecciona la pestaña *From Tutorial*.
4. Selecciona el "Movement tutorial"
5. Selecciona una ubicación para el proyecto en tu disco local y haz click en *Create New Project*.

![new project](images/new-movement.png)

El editor abre automáticamente el archivo "README" desde la raíz del proyecto, que contiene el texto completo del tutorial.

![icon](images/icon-tutorial.svg){.icon} [También puedes leer el texto completo del tutorial en Github](https://github.com/defold/tutorial-movement)

Si te quedas atascado, visita el [Defold Forum](//forum.defold.com), donde recibirás ayuda del equipo de Defold y de muchos usuarios amables.

¡Feliz Defolding!

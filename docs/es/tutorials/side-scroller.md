# Tutorial Side scroller


El tutorial está integrado con el editor Defold y es fácil acceder a él:

Inicia Defold y:
1. Selecciona `Create From` ▸ `TUTORIALS` a la izquierda.
2. Selecciona el "Side Scroller Tutorial".
3. Renómbralo, si quieres.
4. Selecciona una ubicación para el proyecto en tu disco local.
5. Haz click en `Create New Project`.

![new project](images/new-side-scroller.webp)

El editor abre automáticamente el archivo "README" desde la raíz del proyecto, que contiene el texto completo del tutorial.

![icon](images/icon-tutorial.svg){.icon} [También puedes leer el texto completo del tutorial en Github](https://github.com/defold/tutorial-side-scroller)

Si te quedas atascado, visita el [Defold Forum](//forum.defold.com), donde recibirás ayuda del equipo de Defold y de muchos usuarios amables.

# Soporte de Xbox

Defold todavía no es compatible con Microsoft Xbox. La Defold Foundation está trabajando activamente para convertirse en proveedor de Middleware de Xbox. Animamos a los desarrolladores de Xbox aprobados a contactar con su contacto de desarrollador de Xbox y solicitar soporte de Defold para Microsoft Xbox.

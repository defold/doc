# Depuración

Defold contiene un depurador Lua integrado con una herramienta de inspección. Junto con las [herramientas de profiling](/manuals/profiling) integradas, es una herramienta potente que puede ayudar a encontrar la causa de errores en la lógica de tu juego o a analizar problemas de rendimiento. Defold también proporciona logs de crash para las raras ocasiones en que el propio motor falla.

* Aprende más sobre [depurar la lógica de tu juego](/manuals/debugging-game-logic).
* Aprende más sobre [depurar código nativo](/manuals/debugging-native-code).

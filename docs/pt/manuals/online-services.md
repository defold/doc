# Serviços de jogo

Usar requisições HTTP e conexões por socket permite conectar-se e interagir com milhares de serviços diferentes na internet, mas na maioria dos casos há mais envolvidos do que simplesmente fazer uma requisição HTTP. Normalmente você precisa usar algum tipo de autenticação, os dados da requisição talvez precisem ser formatados de uma determinada maneira e a resposta talvez precise ser analisada antes de poder ser usada. É claro que você pode fazer isso manualmente, mas também existem extensões e bibliotecas para cuidar desse tipo de coisa para você. Abaixo você encontrará uma lista de algumas extensões que podem ser usadas para interagir mais facilmente com serviços de backend específicos:

## Uso geral
* [Colyseus](https://defold.com/assets/colyseus/) - Cliente de jogo multiplayer
* [Nakama](https://defold.com/assets/nakama/) - Adicione autenticação, matchmaking, analytics, cloud save, multiplayer, chat e muito mais ao seu jogo
* [Photon Realtime](https://defold.com/assets/photon-realtime/) - Photon Realtime oferece soluções escaláveis para recursos essenciais como autenticação, matchmaking e comunicação rápida e confiável.
* [PlayFab](https://defold.com/assets/playfabsdk/) - Adicione autenticação, matchmaking, analytics, cloud save e muito mais ao seu jogo
* [AWS SDK](https://github.com/britzl/aws-sdk-lua) - Use Amazon Web Services dentro do seu jogo

## Autenticação, placares, conquistas
* [Google Play Game Services](https://defold.com/assets/googleplaygameservices/) - Use Google Play Game Services para autenticar e usar cloud save no seu jogo
* [Steamworks](https://defold.com/assets/steamworks/) - Adicione suporte ao Steam ao seu jogo
* [Apple GameKit Game Center](https://defold.com/assets/gamekit/)

## Analytics
* [Firebase Analytics](https://defold.com/assets/googleanalyticsforfirebase/) - Adicione Firebase Analytics ao seu jogo
* [Game Analytics](https://gameanalytics.com/docs/item/defold-sdk) - Adicione GameAnalytics ao seu jogo
* [Google Analytics](https://defold.com/assets/gameanalytics/) - Adicione Google Analytics ao seu jogo

Confira o [Portal de Assets](https://www.defold.com/assets/) para ainda mais extensões!

# Primeiros passos

A engine e o editor Defold são ferramentas poderosas, com muitos recursos e funcionalidades para aprender. Para ajudar você nesse caminho, criamos vários tutoriais. Muitos deles estão disponíveis diretamente dentro do editor, o que torna muito fácil começar.

## Iniciando um tutorial pelo editor

Ao executar o editor Defold, você verá uma tela de seleção e criação de projetos. A partir dela, é fácil escolher qual tutorial você quer experimentar:

1. Inicie o Defold.
2. Selecione *New Project* à esquerda.
3. Selecione a aba *From Tutorial*.
4. Selecione um tutorial que seja do seu interesse.
5. Selecione uma localização para o projeto no seu disco local.
6. Clique em *Create New Project*.

![create project](images/getting-started/new-project.png)

Agora o editor baixa automaticamente o projeto do tutorial, abre-o no editor e abre o texto do tutorial (o arquivo "README" na raiz do projeto).

![tutorial text](images/getting-started/tutorial-text.png)

Agora siga o texto do tutorial! Se quiser voltar ao texto, dê <kbd>double click</kbd> no arquivo "README" na visualização *Assets*. Você também pode clicar com <kbd>right click</kbd> na aba de arquivos abertos e selecionar <kbd>Move to Other Tab Pane</kbd> para visualizar o texto do tutorial lado a lado com o arquivo em que está trabalhando.

![side by side](images/getting-started/side-by-side.png)

Se você é totalmente novo no Defold, talvez também queira conferir a [introdução ao editor](/manuals/editor).

Se algum dia você ficar preso, acesse o [Defold Forum](//forum.defold.com), onde receberá ajuda dos desenvolvedores do Defold e de muitos usuários prestativos.

Boas criações com Defold!

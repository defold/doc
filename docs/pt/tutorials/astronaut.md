# Tutorial de movimento top-down

Neste tutorial para iniciantes, você aprende a definir e capturar a entrada do jogador e fazer um personagem se mover e animar. Você também conhecerá os blocos de construção centrais do Defold: objetos de jogo, componentes e coleções.

Neste tutorial para iniciantes, você aprenderá a criar um controlador simples de personagem top-down no Defold. Você começará a partir de um projeto preparado especialmente, então não precisará se preocupar com os assets nem com a configuração básica; em vez disso, seu foco será a mecânica.

Você explorará a estrutura básica de um projeto Defold, incluindo coleções, objetos de jogo, sprites, scripts, tilemaps, atlases e câmeras. Você adicionará animações de caminhada ao personagem, tratará a entrada do jogador, moverá o objeto de jogo usando Lua, normalizará o movimento diagonal e trocará as animações com base na direção do movimento.

Ao final, você terá um personagem animado funcional que se move por um pequeno nível em tilemap e estará pronto para expandi-lo com recursos como controles WASD, câmera seguindo o personagem ou um mapa maior.

## Execute diretamente pelo Defold

O tutorial é integrado ao editor Defold e pode ser acessado facilmente pela tela de boas-vindas do Defold:

1. Selecione *Create From* -> <kbd>Tutorials</kbd> à esquerda.
2. Selecione <kbd>Top-down Movement Tutorial</kbd>.
3. Digite um *Title* para o seu projeto.
4. Selecione uma *Location* para o projeto no seu disco local.
5. Clique em <kbd>Create New Project</kbd>.

![new project](images/top-down-start.webp)

O editor abre automaticamente o arquivo "README" na raiz do projeto, que contém o texto completo do tutorial para você seguir.

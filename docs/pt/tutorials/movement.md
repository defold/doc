# Tutorial de movimento

Neste tutorial para iniciantes, você aprende a criar movimento realista usando vetores e um pouco de álgebra vetorial simples.

O tutorial é integrado ao editor Defold e pode ser acessado facilmente:

1. Inicie o Defold.
2. Selecione *New Project* à esquerda.
3. Selecione a aba *From Tutorial*.
4. Selecione "Movement tutorial"
5. Selecione uma localização para o projeto no seu disco local e clique em *Create New Project*.

![new project](images/new-movement.png)

O editor abre automaticamente o arquivo "README" na raiz do projeto, que contém o texto completo do tutorial.

![icon](images/icon-tutorial.svg){.icon} [Você também pode ler o texto completo do tutorial no Github](https://github.com/defold/tutorial-movement)

Se você ficar preso, acesse o [Defold Forum](//forum.defold.com), onde receberá ajuda da equipe do Defold e de muitos usuários prestativos.

Boas criações com Defold!
